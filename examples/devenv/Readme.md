# Jix vs. devenv

Here, we are comparing between Jix and devenv.

### Jix version

In `./jix/` run

```bash
jix run shell
```

See [`./jix/__jix__.js`](./jix/__jix__.js)


### devenv version

In `./devenv/` run
```bash
devenv shell
```

See [`./devenv/devenv.nix`](./devenv/devenv.nix) and other files in [`./devenv`](./devenv).

This devenv example was adapted from https://github.com/cachix/devenv/blob/main/examples/scripts/devenv.nix.
